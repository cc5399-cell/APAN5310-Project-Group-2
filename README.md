# APAN5310 Project Group 2 - ABC Foodmart
***Group Members:***  Olivia Chou, Yike Meng, Afeefah Moazzam, Maggie Teng  

***Description:***  A relational database solution for a NYC grocery chain expanding to 5 locations. Replaces spreadsheets with a normalized PostgreSQL system (16 tables, 3NF), Python ETL pipeline, 10 analytical SQL procedures, and a live Metabase executive dashboard — giving analysts and managers real-time access to sales, inventory, and payroll data.
